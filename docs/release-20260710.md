# MauriResults stabilization release

Validated scope:

- Uploaded exam search uses published metadata and ranked views.
- Database ranks are preserved in search, ranking, and topper views.
- Uploaded concours supports guided wilaya, moughataa, centre, and candidate-number search.
- Uploaded analytics and toppers use the dynamic dashboard API.
- Year availability is derived from active published exams; 2026 remains locked until an active 2026 exam exists.
- Year pages display only active exams for the selected year.
- Unmapped or empty result fields are hidden.
- Shared result links preserve source and candidate parameters.
- Homepage, responsive layout, trust badges, help steps, FAQ, loading, empty, and dark-mode states were polished.
- Vercel and Netlify preview checks passed for the release head.

Database requirement:

Apply `supabase/migrations/20260711_uploaded_results_dashboard_validation.sql` in Supabase before publishing a newly uploaded exam.
