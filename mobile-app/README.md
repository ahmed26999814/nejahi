# MauriResults Mobile

تطبيق Android أصلي مستقل مبني بـ React Native وExpo. لا يستخدم WebView، ويتصل بواجهات MauriResults العامة لعرض النتائج والإحصائيات.

## المزايا في 2.1.0

- واجهة هاتف جديدة وتنقل بخمس تبويبات.
- معالجة زر الرجوع في Android والتنقل داخل التطبيق قبل الخروج.
- ترتيب النتائج حسب الأقرب للبحث أو المعدل أو الاسم أو الرقم.
- حاسبة معدل موزون تحفظ المواد والمعاملات على الهاتف.
- دروس وملخصات مراجعة تعمل دون اتصال.
- لوحة واضحة لحالة نتائج 2025 وتفعيلها تلقائيًا فور نشرها في قاعدة البيانات.
- بحث سريع بالاسم أو رقم المترشح.
- حفظ النتائج وسجل البحث محليًا.
- مشاركة النتيجة وتصديرها PDF.
- إحصائيات المسابقات والأوائل والولايات.
- العربية والفرنسية والوضع الفاتح والداكن.
- لا شعب للكونكور وأبريفه، وترتيب شعب الباك SN ثم M ثم LO ثم LM.

## التشغيل المحلي

```bash
npm install
python3 -m pip install pillow
npm run assets
npm start
```

## إنشاء APK Release

```bash
npm run prebuild
cd android
./gradlew assembleRelease
```

الإصدار: `2.1.0` — الحزمة: `com.mauriresults.app` — versionCode: `3`.
